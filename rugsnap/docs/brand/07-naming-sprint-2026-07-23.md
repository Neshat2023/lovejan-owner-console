# 07 — Naming Sprint (international-first) — 2026-07-23

**Method (corrected):** every candidate screened live on three channels — registrar availability (Vercel API), keyword web search, and direct-domain visit. International-first: a name must work in English/globally, then pass a Farsi no-negative-meaning check. No purchases made.

## What was run
**80 candidate .coms checked live** across territories: value/worth (val-, worth-, merit-), trust/verify (ver-, attest-, verita-), provenance (proven-, proven-oire), objects/curio, coined 2-syllable brandables (Stripe/Ramp style), and Latinate 3-syllable coinages. Batches and raw results are in this session's tool log (2026-07-23 ~22:0x–22:5x UTC).

## Result — the iron law
**In the $12 .com namespace, availability and quality are inversely correlated.** Every short, spellable, premium, brandable `.com` was already registered (val*, ver*, trov*, prov*, object*, veritas, appraisa, valora, valoro, curato, vaulta, lumira, trovato… all taken). The only `.com`s available at registration were:

| Available .com ($11.25) | Meaning | Fatal-ish flaw |
|---|---|---|
| provenoire.com | provenance + -oire (French) | very hard to spell from hearing; pronunciation ambiguous |
| valoresce.com | value + -esce ("coming into being") | mouthful; spelling-from-hearing hard |
| meritvane.com | merit + vane | "vane" reads odd; weak |
| asetjan.com | asset + jan (جان, "dear") | aset/asset one-vs-two-s; "jan" opaque to non-Persian global market |
| asetbaz.com | asset + baz (باز, dealer/enthusiast) | "-baz" roguish undertone in FA; opaque globally |

Available on .ai only (no clean .com): valinto.ai, objevo.ai (both weak/meaningless).

## Conclusion (professional)
A great global `.com` is an **acquisition, not a registration.** The realistic paths, and the honest trade-offs:

1. **Broker a premium .com** (budget path — recommended for the global ambition). Set a budget (FD3: e.g. $2k / $10k / $50k tiers); engage a broker at the trigger to buy a clean one-word or coined `.com` from its owner, alongside the attorney knockout. This is how venture-scale brands actually get their name. *This environment cannot negotiate or verify aftermarket prices — owner-driven.*
2. **Register a flawed-but-available .com now** ($12). Best available = **Asetjan** — and note it is a *bilingual bridge* (English ear hears "asset"; Persian ear hears warmth + the LoveJan family), which makes it more international than pure-Persian or the hard-to-spell coinages. Accept the spelling-mitigation (also register assetjan.com) and the "jan" pronunciation teaching cost.
3. **Launch on a non-.com** (.ai/.app) for an international coined name — rejected for a trust/commerce brand: a squatter on the matching `.com` leaks type-in traffic and undercuts credibility (documented in brand/04).

## Recommendation
- **Do not settle for a spelling-hard .com** (provenoire/valoresce/meritvane) — they fail our hard rejection rules; availability is not a reason to adopt.
- **If acquiring soon:** path 1 (broker + budget + attorney) — the only route to a truly global brand `.com`.
- **If registering cheaply today as a hedge:** path 2 (Asetjan + assetjan defensive, ~$32) — a low-regret hold, not a final commitment.
- Either way, RugSnap keeps running; the umbrella name is still gated to the trigger. Attorney knockout (FD4) precedes any public adoption.

**Method lesson (permanent):** stop spraying $12 checks — 80 confirmed the namespace is exhausted; the constraint is budget, not creativity.
